# XG5000 Shortcuts (LS ELECTRIC PLC)

> XG5000 **기본(Default) 단축키** 기준입니다. XG5000의 **[도구] → [단축키]** 메뉴에서 사용자화할 수 있습니다.
> 심볼 입력키는 XG5000 LD(래더) 편집 매뉴얼 기준입니다.

## ⭐ LD 심볼 입력 (핵심)

| 설명 (Description) | 단축키 (Shortcut) |
|:---|:---|
| 평상시 열린 접점 `─┤├─` | `F3` |
| 평상시 닫힌 접점 `─┤/├─` | `F4` |
| 양변환 검출 접점 `─┤P├─` | `Shift + F1` |
| 음변환 검출 접점 `─┤N├─` | `Shift + F2` |
| 코일 `─( )─` | `F9` |
| 셋(Set) 코일 `─(S)─` | `Shift + F3` |
| 리셋(Reset) 코일 `─(R)─` | `Shift + F4` |
| 양변환 검출 코일 `─(P)─` | `Shift + F5` |
| 응용명령 (Instruction) | `F10` |

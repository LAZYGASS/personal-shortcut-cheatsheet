# PowerPoint Shortcuts (Windows)

## ⭐ 핵심 단축키 (Essential)

| 설명 (Description) | 단축키 (Shortcut) |
|:---|:---|
| 새 슬라이드 추가 | `Ctrl + M` |
| 저장하기 | `Ctrl + S` |
| 작업 취소 (Undo) | `Ctrl + Z` |
| 다시 실행 (Redo) | `Ctrl + Y` |
| 복사 / 붙여넣기 | `Ctrl + C/V` |
| 슬라이드/개체 복제 | `Ctrl + D` |
| 찾기 | `Ctrl + F` |
| 바꾸기 | `Ctrl + H` |
| 처음부터 슬라이드쇼 | `F5` |
| 현재 슬라이드부터 쇼 | `Shift + F5` |
